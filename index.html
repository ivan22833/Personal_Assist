Персональний помічник

Це консольний додаток на Python, який допомагає додавати контакти та нотатки, а також відображати їх у терміналі.

Функціональність

  • Додавання контактів
  • Додавання нотаток
  • Перегляд усіх контактів і нотаток

Як запустити

Для запуску необхідно завантажити архів з github та запустити командою ( docker build -t personal_assist . ) та потім ( docker run --rm -it personal_assist ) .
